The home of the Sky Map Generator

SMG is a program that generates a map of the sky and plots it to a file in PostScript (EPS) format.
It can plot several of the main objects visible in the sky (stars, galaxies, open and globular star clusters, nebulae).
It also contains algorithms for intelligent text positioning so that there are as few overlaps as possible.

It is written in standard C++ which is, except for the Win32 GUI program, heavily relying on templates and the STL.
For viewing the files generated with SMG you will need a PostScript interpreter such as GhostScript.
